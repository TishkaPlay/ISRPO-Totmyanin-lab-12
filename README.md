# Лабораторная работа №12. Введение в Bootstrap

## Описание
Данная лабораторная работа посвящена изучению CSS-фреймворка Bootstrap 5. В ходе выполнения были освоены система сеток (Grid System), готовые компоненты (Navbar, Cards, Forms) и основы работы с Git/GitHub.

## Цель работы
- Познакомиться с CSS-фреймворком Bootstrap 5.
- Освоить систему сеток (Grid System).
- Научиться использовать готовые компоненты Bootstrap.
- Закрепить навыки работы с Git (init, add, commit, push).

## Технологии
- **HTML5**
- **CSS3** (Bootstrap 5.3.8 via CDN)
- **JavaScript** (ванильный JS для валидации форм и фильтрации галереи)
- **Git** (система контроля версий)
